Making an event

When making an event, think about all the things you want to tell people about it. Make sure the event name is short but descriptive, make sure the address is understandable to people living in your city. Make sure the time says whether it's AM or PM, and the date uses mm/dd/yyyy instead of "Today" or "Tomorrow".

Also, make sure the description is fully descriptive of what you're planning. Let people know how much fun they're going to have!
